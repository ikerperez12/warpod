# Warpod Studio ✦ Next-Gen UI/UX Experience

![Warpod Studio](public/assets/images/premium_bg_1.jpg)

**Warpod Studio** is a high-end digital experience crafted with cutting-edge web technologies. This repository showcases a professional design studio landing page, featuring immersive 3D interactions, cinematic scroll storytelling, and ultra-crisp 4K visual assets.

## 🚀 Tech Stack

*   **Framework:** [React 19](https://react.dev/)
*   **3D Engine:** [Three.js](https://threejs.org/) via [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)
*   **Motion:** [GSAP](https://greensock.com/gsap/) & [ScrollTrigger](https://greensock.com/scrolltrigger/)
*   **Smooth Scroll:** [Lenis](https://lenis.darkroom.engineering/)
*   **Styling:** PostCSS & Modern CSS Variables
*   **Build Tool:** [Vite](https://vitejs.dev/)

## ✨ Key Features

*   **The Architectural Forge:** An immersive storytelling section where the tech stack is assembled in real-time 3D.
*   **Kinetic Core:** A high-fidelity 3D environment with interactive physics reacting to user movement.
*   **Curtain Transitions:** Seamless high-resolution video transitions between major sections.
*   **Fanning Deck Gallery:** A dynamic portfolio showcase with fan-out card animations.
*   **Star Wars Scrub Text:** A cinematic typography experience with fluid multicolor gradients and depth-based scroll animations.

## 🛠️ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/warpod-studio.git
    cd warpod-studio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Launch the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```

## 🔒 Security & Performance

*   **Optimized Assets:** All images and videos are pre-processed for 4K clarity without sacrificing load times.
*   **Memory Management:** Automatic preloading of 3D models and aggressive cleanup of ScrollTrigger instances to ensure 60FPS performance.
*   **Clean Repository:** No external dependencies or project inheritance. A strictly focused codebase ready for public deployment.
